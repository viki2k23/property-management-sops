# Applicant Screening Checklist

## Objective

Review rental applicants using a consistent and fair process.

## Basic Information

- Full legal name
- Phone number
- Email address
- Desired move-in date
- Number of occupants

## Financial Review

- Monthly income
- Income verification documents
- Employment status
- Debt concerns if disclosed

## Rental History

- Previous landlord reference
- Payment history
- Prior evictions if legally reportable
- Length of previous tenancy

## Identity Verification

- Government ID
- Application consistency check

## Decision Notes

- Approved
- Conditional approval
- Need more documents
- Declined (per policy and applicable laws)

## Record Keeping

Store application notes and supporting documents securely.
